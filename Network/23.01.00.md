# 용어

### 프로토콜
- 통신규약
- 물리적 : 이더넷, 랜선 등
- 논리적 : TCP/IP, SMTP 등
<br><br>

### 각 계층별 전송 기술
- 5~7계층 : 애플리케이션 계층
- 4계층 : 전송 계층, 프로세스를 구분
- 3계층 : 네트워크 계층(인터넷), 호스트(컴퓨터)를 구분
- 1~2계층 : 피지컬/데이터 링크(네트워크 액세스) 계층, 노드를 구분
<br><br>

### 1계층 (피지컬)
<pre>전기 신호 전달</pre>
- 허브, 리피터 : 네트워크 통신을 중재하는 네트워크 장비
- 케이블, 커넥터 : 케이블 본체 구성 요소
- 트랜시버 : 컴퓨터 랜카드와 케이블 연결 장비
- 탭 : 네트워크 모니터링과 패킷 분석을 위해 전기 신호를 다른 장비로 복제
